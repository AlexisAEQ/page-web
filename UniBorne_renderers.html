Interface de configuration UniBorne

Passez votre souris sur les éléments de l'interface pour voir leur fonction.

RxLabel Affiche une donnée simple en lecture seule (texte ou booléen) provenant de l'automate.
TxString Permet à l'opérateur de saisir du texte et de l'envoyer vers la machine (ex: nom de lot).
RxIndicator Voyant lumineux d'état. Change de couleur (Vert/Rouge/Gris) selon l'état d'une variable.
TxDropdown Menu déroulant permettant de choisir une option parmi une liste pré-configurée.
RxLevelBar Visualisation graphique d'une valeur analogique (jauge de niveau, température, pourcentage).
RxBool Interrupteur visualisant l'état d'une entrée numérique.
TxQr Génère un code QR dynamique ou permet de scanner un code via la caméra de la tablette.
TxBool Interrupteur de commande (Switch) pour activer ou désactiver une sortie machine.
TxButtonHold Bouton momentané : la variable reste active tant que l'utilisateur maintient l'appui (fonction "jog").
TxNumeric Champ de saisie pour envoyer une valeur numérique précise (consigne) à l'automate.
TxSlider Curseur glissant pour ajuster rapidement une valeur analogique entre 0 et 100%.
TxModal Bouton déclenchant l'ouverture d'une fenêtre modale (pop-up) ou d'une page protégée.
